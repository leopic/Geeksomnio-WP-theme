== Description ==

Want to switch between two themes according to a specific criteria in your URI?

Just add and activate this small plugin !

More at <a href="http://agafix.org/how-to-automatically-switch-wordpress-themes-onthefly/" target="_blank">http://agafix.org/how-to-automatically-switch-wordpress-themes-onthefly/</a>

== Installation ==

1. Upload `wp-theme-switcher` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Pattern rules in the admin page
